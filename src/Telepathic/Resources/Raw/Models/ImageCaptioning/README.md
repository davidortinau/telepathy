# Image Captioning Models

This directory should contain the ONNX models for image captioning using ViT-GPT2.

## Required Files

Download from: https://huggingface.co/Xenova/vit-gpt2-image-captioning/tree/main

Required files:
- encoder_model.onnx
- decoder_model_merged.onnx  
- tokenizer.json

## Build Configuration

Ensure these files have Build Action = Content and Copy to Output Directory = Copy Always
