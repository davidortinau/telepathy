# Voice Transcription Models

This directory should contain the ONNX models for voice transcription using Whisper-tiny.

## Required Files

Download from: https://huggingface.co/openai/whisper-tiny/tree/main

Required files:
- encoder_model.onnx
- decoder_model_merged.onnx
- tokenizer.json

## Build Configuration

Ensure these files have Build Action = Content and Copy to Output Directory = Copy Always
